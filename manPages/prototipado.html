2.1 - Prototipado

Las clases en JavaScript no son iguales a las de Java. Pero gracias a la flexibilidad de JavaScript se pueden emular.

¿Qué es una clase a grandes rasgos? Un esquema que confiere un comportamiento propio a un objeto concreto. Todo ello en JavaScript se puede conseguir con prototype.

Uno de los paradigmas de JavaScript es el de orientado a objetos, aunque aquí hay que resaltar muy bien que se basa en la tradicional programación orientada en objetos, sino en la programación basada en prototipos.

¿Qué es un prototipo? Es un objeto con un estado concreto del cuál otros nuevos objetos pueden clonar. Cualquier prototipo de cualquier objeto puede ser clonado, lo que abre grandes posibilidades y grandes problemas. "Una gran poder conlleva una gran responsabilidad".

Qué ventajas ofrece:

  • El comportamiento de un prototipo puede modificarse durante la ejecución. Lo que implica que cualquier objeto que referencie dicho prototipo cambiará su comportamiento en consonancia.
  • Al ser un objeto en sí, puede ser nombrado y llamado.
  • Todo objeto puede tener sus atributos y métodos propios, aquellos de su prototipo se heredan. Si heredan, la jerarquía de prototipos puede entenderse como herencia.
var Persona = function() {
	this.especie = "Ser humano";
	this.nombre = "";
	this.diHola = function() {
		console.log("Soy una persona y me llamo", this.nombre);
	};
};

var Mujer = function(nombre) {
	this.nombre = nombre;
	this.diHola = function() {
		console.log("Soy una mujer y me llamo", this.nombre);
	};
};

/* Ahora existen dos objetos distintos, para que Mujer herede el
comportamiento de Persona es necesario referenciar Persona como su prototipo. */
Mujer.prototype = new Persona();

// Creamos un objeto persona, cuyo prototipo es Object.
var pedro = new Persona();

// Creamos un objeto mujer, cuyo prototipo es Persona (cuya clase base es Object).
var ana = new Mujer("Ana");

pedro.diHola(); // Resultado: Soy una persona y me llamo (string vacío)

ana.diHola();   // Resultado: Soy una mujer y me llamo Ana

ana.especie;    // Resultado: "Ser humano"
