Construcción de una clase básica

Una clase es un conjunto de propiedades y métodos.

  • Propiedades: variables que sirven para establecer el estado del objeto.
  • Métodos: conjunto de funciones que definen el comportamiento del objeto.

Definir una clase es muy sencillo en Javascript.

var Persona = function() {
	// Propiedades (ya sean constantes, enumeraciones o variables)
	this.especie = "Ser humano";
	this.nombre  = undefined;
	this.edad    = undefined;
};

Persona.prototype = {
	getEspecie: function() {
		return this.especie;
	},
	sayHello: function() {
		console.log("Hola!");
	}
};

var juan = new Persona();
juan.nombre = "Juan";

console.log(juan);
// Resultado: Object { especie="Ser humano", nombre="Juan", edad=undefined, more...}

Método constructor

Toda clase que se precie necesita un método constructor que es el encargado de construir adecuadamente un nuevo objeto.

La libertad de JavaScript permite hacerlo de cualquier forma, sin embargo, ésta es la más correcta:

var Persona = function() {
	// Propiedades (ya sean constantes, enumeraciones o variables)
	this.especie = "Ser humano";
	this.nombre  = undefined;
	this.edad    = undefined;

	// Es obligatorio llamar al método constructor con los argumentos.
	this.constructor.apply(this, arguments);
};

Persona.prototype = {
	/**
	 * Siempre que se crea un nuevo objeto este método es llamado.
	 * @constructor
	 */
	constructor: function(nombre, edad) {
		this.nombre = nombre;
		this.edad   = edad;
	},

	getEspecie: function() {
		return this.especie;
	},

	sayHello: function() {
		console.log("Hola!");
	}
};

var juan = new Persona("Juan", 25);
juan.especie = "Alien?";

console.log(juan);
// Resultado: Object { especie="Alien?", nombre="Juan", edad=25, more...}
