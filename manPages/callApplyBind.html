1.1 - Modificar el scope de una función

Como hemos visto antes, la variable this referencia al scope existente en cada momento de la lógica del programa. Este scope varía cuando se produce un cambio en el contexto, por ejemplo, cuando se ejecuta una función determinada. ¿Es posible manipular esta referencia a nuestro gusto? La respuesta es si.

Existen tres funciones que manipulan el scope aunque se puede reducir el número a dos o incluso a una única función.

  • bind(scope[, arg1, arg2, ...]): crea una función nueva con el scope y argumentos preestablecidos.
  • call(scope[, arg1, arg2, ...]): ejecuta la función con el scope y los argumentos suministrados. A grandes rasgos, el comportamiento de call es parecido a bind(...)(), es decir, si acto seguido de hacer el bind se ejecuta la función.
  • apply(scope[, arrayArgs]): su comportamiento es igual que call, sin embargo, los argumentos se deben pasar como un array.

Esta capacidad de manipular el scope abre muchas posibilidades y usos avanzados. Si no se especifica el scope, entonces se define como aquel más general.

var juan = { nombre: "juan" },
    jose = { nombre: "jose" };

function capitalize(string) {
	return string.charAt(0).toUpperCase()+string.slice(1);
}

capitalize(juan.nombre); // Resultado: "Juan"

jose.getCapitalizedName = capitalize.bind(jose, jose.nombre);

jose.getCapitalizedName(); // Resultado: "Jose"
